## Powerful Features

<CardGrid>
  <Card title="Compound Variants" icon="setting">
    Apply styles when multiple variant conditions are met simultaneously.

    ```typescript
    compoundVariants: [
      {
        color: 'primary',
        size: 'lg',
        class: 'uppercase tracking-wide'
      }
    ]
    ```
  </Card>
  <Card title="Boolean Variants" icon="approve-check-circle">
    Simple on/off toggles for common states like disabled, loading, or active.

    ```typescript
    variants: {
      disabled: {
        true: 'opacity-50 cursor-not-allowed'
      }
    }
    ```
  </Card>
  <Card title="Slot System" icon="list-format">
    Style multi-element components with named slots for complex UI patterns.

    ```typescript
    slots: ['root', 'header'],
    base: {
      root: 'card-root',
      header: 'card-header'
    }
    ```
  </Card>
  <Card title="Class Merging" icon="random">
    Smart class name merging with the `cx` utility. Handle conditional classes with ease.

    ```typescript
    cx('base', condition && 'active', {
      'enabled': isEnabled
    })
    ```
  </Card>
</CardGrid>
